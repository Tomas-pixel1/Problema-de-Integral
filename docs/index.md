# Problema

Se quiere resolver la siguiente integral:

$\int_{0}^{\pi}sin(x^2)\ dx$

El problema es que esta integral no tiene una solución analítica. Para esto se usan los métodos numéricos. Por medio de ellos, se puede aproximar la solución. Para este caso, se usará un método llamado cuadratura gaussiana para poder aproximar la solución de la integral anteriormente mencionada.
